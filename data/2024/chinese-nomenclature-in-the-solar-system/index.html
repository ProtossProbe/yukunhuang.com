太阳系天体的中文命名及术语

越来越多的华人天文学家以及行星科学家都在研究太阳系内小天体。但许多被国际天文学联合会（IAU）命名的小天体缺乏公认的中文翻译，这让华人学者间的交流以及向公众的科普变得困难。故编此列表，以收录太阳系内小天体的中文译名。 对于在 天文学名词 中收录了的天体，我会提供出处（例如，冥王星）；对于尚未被广泛采用的翻译，我将列出大众所熟知的名称或自己的译名，并用双引号标出（如 “振荡天星”）。

An increasing number of Chinese astronomers and planetary scientists are conducting research on small bodies within the Solar System. However, many celestial bodies that have been officially named by the International Astronomical Union (IAU) lack widely recognized Chinese translations. This creates challenges for communication among Chinese-speaking researchers as well as for public outreach. Therefore, I have decided to compile this list to record the Chinese translations of Solar System objects. For celestial bodies whose translations have already been included in the Astronomical Terms Dictionary, their sources will be provided (e.g., 冥王星). For translations that are not yet widely adopted, commonly known names will be listed or I will provide my own translations, which will be enclosed in quotation marks (e.g., “振荡天星”).

地球轨道以内

  • Zoozve (2002 VE68)

    ⟡ “准金卫星”

    → Zoozve 来源于 Latif Nasser 对于小行星临时编号 “2002VE” 的误读。该小行星是金星的一颗”拟卫星”或”准卫星”，故提议译为”准金卫星”，其中”准”与 Zoo，”卫”与 ve 的发音类似。

  • ꞌAylóꞌchaxnim (2020 AV2)

    ⟡ “爱洛查赫妮姆”

    → 第一颗被发现轨道完全位于金星轨道内的小行星。在南加州的土著路易塞尼奥语中的意思是”金星女孩”。

近地空间

  • Kamoʻoalewa (2016 HO3)

    ⟡ “振荡天星”

    → 地球的拟卫星，其名称来源于夏威夷语”振荡的天星”，亦是中国天问二号小行星采样任务的目标小行星。使用此名的相关新闻报道。

  • Itokawa (1998 SF36)

    ⟡ 丝川/糸川

    → JAXA 隼鸟一号任务的目标小行星，其名称来源于纪念日本火箭专家糸川英夫。

  • Ryugu (1999 JU3)

    ⟡ 龙宫

    → JAXA 隼鸟二号任务的目标小行星，其名称来源于日本民间传说中海龙王的住所。

  • Bennu (1999 RQ36)

    ⟡ 贝努

    → NASA OSIRIS-REx 任务的目标小行星，其名称来源于埃及神话的不死鸟。

  • Didymos

    ⟡ 孪大星

    → NASA 双小行星重定向测试任务（DART）的目标双小行星的主星。

  • Dimorphos

    ⟡ 孪小星

    → 孪大星的卫星。

  • Apophis (2004 MN4)

    ⟡ 毁神星/“阿波菲斯”

    → 将会在 2029 年 4 月 13 日近距离飞越地球的小行星。在发现之初曾被认为有概率撞上地球而得此名。

小行星带

  • Ceres

    ⟡ 谷神星

    → 罗马神话中的掌管农业与谷物的女神。

  • Psyche

    ⟡ 灵神星

    → 希腊神话中的女神，是人类灵魂的化身。NASA 灵神星任务（Psyche）的目标小行星。

  • Dinkinesh

    ⟡ “丁基内什”

    → 是阿姆哈拉语中对露西化石的称呼。

  • Ao

    ⟡ “苍”

    → 真中苍是日本漫画家 Quro 所作动漫《恋爱小行星》中的角色。苍与她的朋友怀着将一颗小行星命名为“苍”的梦想，在高中积极进行地球科学社的活动。该作凭借著对天体与地质学知识的正确描绘，激发了大量读者研究地学与行星科学的兴趣。

木星共轨空间

  • Kaʻepaokaʻāwela (2015 BZ509)

    ⟡ “逆木星”

    → Kaʻepaokaʻāwela 是木星的逆行共轨小行星，其名称来源于夏威夷语”淘气的木星逆行伙伴”，故提议译为”逆木星”。

半人马小行星

  • Chiron

    ⟡ 喀戎

    → 希腊神话中的半人马英雄。

  • Chariclo

    ⟡ ”查理洛“

    → 希腊神话中的喀戎的妻子，阿波罗的女儿。

  • Typhon

    ⟡ 台神星

    → 希腊神话中象征风暴的妖魔巨人。

术语

  • Dwarf Planet - 矮行星
  • TransNeptunian Object (TNO) - 海外天体
  • Kuiper Belt - 柯伊伯带
  • Classical Kuiper Belt Object - 经典柯伊伯带天体
  • Plutino - 冥族天体
  • Scattered/Scattering Disk Object - 散盘型天体
  • Resonant Object - “共振天体”
  • Detached Objects - “离群天体”
  • Sednoid - 类赛德娜天体 / “赛族天体”

冥族天体

  • Pluto

    ⟡ 冥王星

    → 罗马神话中的冥王。

  • Ixion

    ⟡ 伊克西翁

    → 希腊神话中狂妄的拉皮斯人国王。

  • Huya

    ⟡ 雨神星

    → 南美洲原住民瓦尤人神话中的雨神。

  • Lempo

    ⟡ “伦波”/“恶神星”

    → 最初为芬兰神话中的爱和生育之神，在基督教引入芬兰后被描绘成一个魔鬼。

  • Rhadamanthus

    ⟡ 拉达曼提斯

    → 希腊神话中的半神，死后成为冥界判官。

  • Arawn

    ⟡ “亚尔翁”

    → 威尔士神话中的冥界之王。

经典柯伊伯带天体（热）

  • Haumea

    ⟡ 妊神星

    → 夏威夷神话中的生育之神。

  • Makemake

    ⟡ 鸟神星

    → 复活节岛拉帕努伊族原住民神话中的人类创造者与生殖之神。

  • Orcus

    ⟡ 亡神星

    → 古罗马伊特鲁里亚神话的亡者之神。

  • Quaoar

    ⟡ 创神星

    → 南加州的美洲原住民通格瓦部族的创世之神。

  • Salacia

    ⟡ 萨拉客亚

    → 希腊神话中的海洋女神，海王尼普顿的妻子。

  • Varda

    ⟡ 瓦尔达

    → 托尔金作品《精灵宝钻》中维拉的皇后。

  • Varuna

    ⟡ 伐楼那

    → 印度神话中天空，雨水及天海之神。

  • Chaos

    ⟡ 卡俄斯

    → 希腊神话中的原始神混沌。

经典柯伊伯带天体（冷）

  • Albion

    ⟡ 阿尔比翁

    → 英国诗人和画家威廉·布莱克神话中的原始巨人。

  • Arrokoth

    ⟡ “阿洛克斯”

    → 美洲原住民部落波瓦坦语中的含义为“天空”。

  • Deucalion

    ⟡ 丢卡利翁

    → 希腊神话中的普罗米修斯的儿子，大洪水后少数幸存的人类。

  • Logos

    ⟡ 逻各斯

    → 在诺斯底传统中，逻各斯和佐伊是神性的一对化身，并且是该传统创世神话的一部分。

  • Borasisi

    ⟡ 博拉西西

    → 库尔特·冯内古特的小说《猫的摇篮》中虚构的造物主神。在书中，博拉西西是太阳，而帕布是月亮的名字。

共振天体

  • Gonggong

    ⟡ 共工

    → 中国神话中的水神。

散盘型天体

  • Eris

    ⟡ 阋(xì)神星

    → 希腊神话中的不和女神。

  • Gǃkúnǁʼhòmdímà

    ⟡ “雹神星”

    → 纳米比亚萨恩人神话中的土豚少女。她保护她的人民，并以刺条、冰雹雨云，和魔法羚羊角惩罚作恶者。

  • Taowu

    ⟡ 梼杌(táo wù)

    → 中国古代神话中的四大凶首之一，人头虎腿长有野猪獠牙。471325 Taowu（梼杌）是一颗逆行的海王星外天体，它的轨道倾角为 110°，估测直径在 100 千米。2025 年 2 月 3 日，国际天文联合会提议以中国神话中“四凶”之一的“梼杌(táo wù)”来命名该小天体。

类赛德娜天体

  • Sedna

    ⟡ 赛德娜

    → 因努伊特神話中的海洋女神。

  • Leleākūhonua

    ⟡ “徙神星”

    → 第三颗发现的类赛德娜天体，其名称来源于夏威夷语”飞到陆地出现”。此命名来源于参加 A Hua He Inoa 夏威夷语言项目的学生。这颗天体的轨道让学生们想到了从阿拉斯加迁徙到夏威夷的金斑鸻（Pacific Golden Plover）。根据 IAU 的描述，这个名字将轨道比作候鸟的迁徙轨迹，描述了对接近陆地的渴望。考虑到此名称较难音译，且其他海外天体的译名大多遵从”X神星”的格式，提议将此天体译为”徙神星”。

星际天体

  • ʻOumuamua

    ⟡ 奥陌陌

    → 第一颗发现的星际天体，其名称来源于夏威夷语”信使”。
